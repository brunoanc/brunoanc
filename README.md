## 🚀 Projects

### EternalModManager
Cross-platform mod manager for DOOM Eternal written in Rust.

- 4.6k+ downloads via Flathub  
- Concurrent architecture focused on reliability and memory safety  

### SMMUN Cloud Registration System
Cloud-native event registration platform.

- Event-driven architecture on Google Cloud  
- FastAPI backend with Pub/Sub, Firestore and Cloud Storage  
- Infrastructure managed with Terraform
